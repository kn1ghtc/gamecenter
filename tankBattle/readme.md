# Tank Battle Game (坦克大战)

## 简介
本项目是一个基于Pygame开发的2D坦克大战游戏，灵感来自经典游戏《Battle City》。玩家控制绿色坦克，击败红色AI坦克，保护基地。

## 玩法说明
- 玩家坦克：W/S前进/后退，A/D旋转，空格射击，P键暂停。
- 敌方坦克：AI控制，自动移动、旋转并射击，击毁后会重生。
- 子弹：直线飞行，碰撞坦克/墙体/边界后消失。
- 地图：包含障碍物（灰色矩形），阻挡坦克和子弹。
- 基地：保护中央基地，基地被摧毁或玩家生命为0则游戏结束。
- 得分：每击毁一辆敌方坦克得10分。
- 胜利条件：消灭所有敌人或生存5分钟。

## 运行方法
1. 安装依赖：
	```bash
	pip install pygame
	```
2. 启动游戏：
	```bash
	python main.py
	```

## 主要技术点
- Pygame图形与事件处理
- 面向对象设计（Tank, Bullet, Wall, Base, Game类）
- AI简单寻路与攻击
- 碰撞检测与游戏状态管理

当前版本使用Pygame内置绘图，无需外部图片或音效。后续可扩展assets目录，导入高质量坦克/爆炸图片与音效。
