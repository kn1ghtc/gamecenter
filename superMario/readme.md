# 超级玛丽兄弟 (Super Mario Bros)

一个完整的超级玛丽兄弟游戏，使用 Python 和 Pygame 开发。支持 30 个关卡，包含积分系统、声音效果和自动资源下载功能。

## 游戏特色

- 🎮 **30 个精心设计的关卡** - 从简单到困难的渐进式挑战
- 🏆 **完整的积分系统** - 金币收集、敌人击败、时间奖励
- 🔊 **丰富的声音效果** - 跳跃、拾取金币、背景音乐等
- � **3D风格精美图像** - 带渐变和阴影的3D风格角色、砖块、敌人和金币
- 🌏 **中英双语支持** - 完美的中文显示和英文字体回退
- 🖥️ **跨平台兼容** - 支持 Windows 和 macOS 系统
-  **平滑摄像机系统** - 实时跟随玩家,支持多层视差背景
- 🚀 **快速启动** - 所有资源预先集成,无需等待下载

## 系统要求

- Python 3.8+
- Pygame 2.5.0+
- Pillow (PIL) - 图像处理
- Requests - 网络请求
- NumPy - 数值计算
- SciPy - 科学计算（用于音效生成）

## 游戏机制

### 角色系统
- **玛丽 (Mario)**: 主角，可以跳跃和移动
- **超级玛丽 (Super Mario)**: 收集道具后变大，更加坚韧

### 跳跃与平台机制
- **土狼时间**: 允许玩家在离开平台后的短时间内仍能起跳,容错更高
- **跳跃缓冲**: 起跳按键可提前触发,角色落地后立即执行跳跃
- **三连跳**: 支持空中连续按空格键最多3次跳跃(地面首跳+空中2次),极致操作体验
- **踩踏弹跳**: 踩敌人后自动反弹并重置空中跳跃次数

### 道具系统
- **金币 (Coins)**: 收集金币获得积分
- **蘑菇 (Mushroom)**: 让玛丽变大成为超级玛丽
- **敌人 (Enemies)**: 各种敌人，需要跳跃或踩踏击败

### 积分系统
- 金币: 200 分
- 击败敌人: 100 分
- 收集道具: 1000 分
- 关卡完成: 5000 分
- 时间奖励: 剩余时间 × 50 分
- 积分暂存于全局记分系统中，可随时用于展示和统计

### 关卡设计
每个关卡包含：
- 地面和平台
- 金币和道具
- 各种敌人
- 关卡目标

## 开发说明

### 核心架构
- **游戏循环**: 基于 Pygame 的经典游戏循环
- **状态管理**: 菜单、游戏中、暂停、游戏结束等状态
- **组件化设计**: 每个功能模块独立开发和维护

### 资源管理
- **预集成资源**: 所有游戏资源(图像、音频、字体)已预先生成并打包
- **3D风格图像**: 使用PIL生成带渐变、阴影和高光的3D风格精灵
- **跨平台字体**: 自动检测系统并选择合适的字体

### 关卡系统
- **程序化生成**: 关卡可以程序化生成或从文件加载
- **难度递进**: 随着关卡编号增加，难度逐渐提升
- **可扩展性**: 容易添加新的关卡类型和敌人

## 故障排除

### 常见问题

1. **字体显示异常**
   - 确保系统安装了中文字体
   - 游戏会自动尝试多种字体

2. **声音无法播放**
   - 检查音频设备
   - 游戏会跳过无法加载的声音文件

3. **游戏启动缓慢**
   - 所有资源已预集成,如仍缓慢请检查系统性能
   - 首次加载字体系统可能需要几秒钟

4. **游戏运行缓慢**
   - 降低游戏分辨率
   - 关闭不必要的后台程序

### 性能优化
- 使用对象池管理游戏对象
- 优化碰撞检测算法
- 按需加载资源
