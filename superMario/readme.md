# 超级玛丽兄弟 (Super Mario Bros)

一个完整的超级玛丽兄弟游戏，使用 Python 和 Pygame 开发。支持 30 个关卡，包含积分系统、声音效果和自动资源下载功能。

## 游戏特色

- 🎮 **30 个精心设计的关卡** - 从简单到困难的渐进式挑战
- 🏆 **完整的积分系统** - 金币收集、敌人击败、时间奖励
- 🔊 **丰富的声音效果** - 跳跃、拾取金币、背景音乐等
- 🌐 **自动资源下载** - 启动时自动下载所需的游戏资源
- 🌏 **中英双语支持** - 完美的中文显示和英文字体回退
- 🖥️ **跨平台兼容** - 支持 Windows 和 macOS 系统
- 🎨 **精美的视觉效果** - 彩色精灵和动画

## 系统要求

- Python 3.8+
- Pygame 2.5.0+
- Pillow (PIL) - 图像处理
- Requests - 网络请求
- NumPy - 数值计算
- SciPy - 科学计算（用于音效生成）

## 游戏机制

### 角色系统
- **玛丽 (Mario)**: 主角，可以跳跃和移动
- **超级玛丽 (Super Mario)**: 收集道具后变大，更加坚韧

### 道具系统
- **金币 (Coins)**: 收集金币获得积分
- **蘑菇 (Mushroom)**: 让玛丽变大成为超级玛丽
- **敌人 (Enemies)**: 各种敌人，需要跳跃或踩踏击败

### 积分系统
- 金币: 200 分
- 击败敌人: 100 分
- 收集道具: 1000 分
- 关卡完成: 5000 分
- 时间奖励: 剩余时间 × 50 分

### 关卡设计
每个关卡包含：
- 地面和平台
- 金币和道具
- 各种敌人
- 关卡目标

## 开发说明

### 核心架构
- **游戏循环**: 基于 Pygame 的经典游戏循环
- **状态管理**: 菜单、游戏中、暂停、游戏结束等状态
- **组件化设计**: 每个功能模块独立开发和维护

### 资源管理
- **自动下载**: 启动时检查并下载缺失资源
- **后备资源**: 下载失败时创建简单的占位符资源
- **跨平台字体**: 自动检测系统并选择合适的字体

### 关卡系统
- **程序化生成**: 关卡可以程序化生成或从文件加载
- **难度递进**: 随着关卡编号增加，难度逐渐提升
- **可扩展性**: 容易添加新的关卡类型和敌人

## 故障排除

### 常见问题

1. **字体显示异常**
   - 确保系统安装了中文字体
   - 游戏会自动尝试多种字体

2. **声音无法播放**
   - 检查音频设备
   - 游戏会跳过无法加载的声音文件

3. **资源下载失败**
   - 检查网络连接
   - 游戏会创建后备资源继续运行

4. **游戏运行缓慢**
   - 降低游戏分辨率
   - 关闭不必要的后台程序

### 性能优化
- 使用对象池管理游戏对象
- 优化碰撞检测算法
- 按需加载资源

## 已知问题

### 当前版本已知问题
1. **字体文件加载问题**: 中文字体文件可能在某些系统上无法正确加载，游戏会自动回退到系统默认字体
2. **背景音乐缺失**: `background_music.mp3` 文件不存在，游戏将以静音背景音乐模式运行
3. **网络资源下载失败**: 部分外部资源URL可能失效，但游戏会创建功能完整的本地资源

### 解决方案
- 字体问题：游戏会自动使用系统字体，不会影响游戏进行
- 音乐问题：游戏设计为可无背景音乐正常运行
- 资源问题：所有核心资源都通过程序生成，确保功能完整
