# 🏆 Professional Chess Game
## 专业国际象棋游戏系统

一个功能完整的国际象棋游戏，集成了四种不同难度的AI对手，包括最新的**超级智能陪伴助理**，支持实时对弈、语音交互、持久记忆、智能工具调用和机器学习训练。

## 🤖 Chess AI Agent - 超级智能陪伴助理

### ✨ 革命性AI助理体验
基于OpenAI Agents SDK和ChromaDB的新一代象棋陪伴助理，具备四大核心AI能力：

#### 🧠 Memory（记忆系统）
- **ChromaDB向量数据库**: 替代FAISS，使用Langchain集成的ChromaDB和LanceDB双数据库架构
- **语义记忆检索**: 智能记忆棋局、对话、策略经验，支持语义相似度搜索
- **持久化学习**: 记住玩家习惯、棋风特点，不断积累象棋知识和战术经验
- **分层记忆管理**: 短期/中期/长期记忆，自动重要度评估和记忆淘汰

#### 🎯 Planning（规划执行）
- **自主任务规划**: 根据游戏状态和玩家需求自动规划最佳策略
- **优先级管理**: 智能任务分解、优先级排序和执行协调
- **动态适应**: 实时监控游戏发展，动态调整策略和执行计划

#### 🛠️ Tools（工具调用）
- **MCP服务集成**: 深度研究服务器、网络搜索、学术论文查询
- **象棋专业工具**: 位置评估、战术发现、开局理论查询
- **智能工具选择**: 根据需求自动选择最适合的工具进行链式调用

#### 🎙️ Voice（语音交互）
- **跨平台语音支持**: Windows + macOS，智能回退到系统语音引擎
- **OpenAI语音引擎**: TTS-1语音合成，Whisper-1语音识别，30秒超时保护
- **情感化语音合成**: 5种情感类型（友好、严肃、鼓励、兴奋、思考），语音个性匹配
- **语音指令识别**: 支持"开始游戏"、"悔棋"、"分析局面"等核心指令
- **双向语音交互**: 语音输入→文本处理→语音输出的完整对话流程
- **多语音引擎**: OpenAI + 系统TTS双引擎，智能故障回退机制
- **音频缓存管理**: 临时文件自动清理，支持pygame和系统播放器

#### 🎤 语音功能特性
- **实时语音识别**: 支持中英文语音指令，5秒识别超时，智能短语分割
- **语音助手模式**: 在游戏界面点击"🎙️ 助手"按钮进入语音交互模式
- **指令映射系统**: 自然语言到游戏指令的智能转换
- **语音状态反馈**: 录音中、识别中、合成中的视觉状态指示
- **错误处理机制**: 网络超时、API异常、音频设备问题的优雅处理

### 💡 陪伴模式特性
- **四种个性**: 友好导师、竞技专家、耐心教师、休闲朋友
- **主动互动**: AI主动评论局面、提供教学指导和策略建议
- **问答系统**: 回答象棋规则、历史、理论等任何相关问题
