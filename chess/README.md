  - 🧠 **MCTS搜索**: 蒙特卡洛树搜索 + 神经网络引导
  - ⚡ **并行自对弈**: 8进程数据生成，GPU加速训练
  - 📊 **智能优化**: 经验回放、课程学习、对手池
- **模型**: 10.5M参数残差网络，12层深度 + 注意力机制
- **性能**: 混合精度训练，40MB模型大小，实时推理

#### 🔴 HARD - GPT专家AI
- **引擎**: GPT-4o-mini
- **特点**: 世界级象棋水平，深度战略分析
- **功能**: 提供移动解释和位置评估，支持多种响应格式解析
- **要求**: 需要OpenAI API密钥

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Windows/Linux/macOS
- (可选) CUDA支持的GPU用于机器学习训练
- (可选) OpenAI API密钥用于GPT AI

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd chess
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **生成棋子图像**
```bash
python assets/piece_generator.py
```

4. **启动游戏**
```bash
python main.py
```

### 可选配置

**设置OpenAI API** (启用GPT AI):
```bash
# Windows
set OPENAI_API_KEY=your_api_key_here

# Linux/macOS  
export OPENAI_API_KEY=your_api_key_here
```

## 🐛 故障排除

### 常见问题

**1. 棋子图像缺失**
```bash
python assets/piece_generator.py
```

**2. GPT AI无响应**
- 检查API密钥设置
- 验证网络连接
- 查看控制台错误信息

**3. ML AI训练问题**
- **训练缓慢**: 检查CUDA可用性，使用现代训练系统
- **内存不足**: 减少批次大小，使用梯度累积
- **收敛困难**: 调整学习率，使用课程学习

**4. 统一训练系统问题**
```bash
# 检查系统状态
cd training
python trainer.py  # 选择快速验证模式

# 查看训练日志（现代模式）
python trainer.py  # 选择现代架构训练

# 传统模式训练
python trainer.py  # 选择传统架构训练

# 重置模型
rm models/*
```

**5. 依赖安装失败**
```bash
pip install --upgrade pip
pip install -r requirements.txt --no-cache-dir
```
