# Games Directory

This directory is intended to store PGN (Portable Game Notation) files for chess games.

## Usage

- **Export Games**: Games can be exported from the database to PGN format
- **Import Games**: External PGN files can be imported and analyzed
- **Backup**: Store backup copies of important games

## File Format

Standard PGN format files (.pgn) containing chess game records with metadata:
- Player names
- Game date and time
- Game result
- Move sequences
- Position evaluations (if available)

## Future Features

- Automatic game export on completion
- PGN file browser and viewer
- Game analysis and annotation tools
