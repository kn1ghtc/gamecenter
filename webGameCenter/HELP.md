# 网页游戏中心 - 用户帮助指南

## 📚 快速开始

### 启动应用
```bash
cd d:\pyproject\gamecenter\webGameCenter
python app.py
```
应用会在 `http://127.0.0.1:5000` 运行。

### 登录/注册
1. 点击页面右上角的 **登录** 按钮
2. 如果没有账户，点击 **注册新账户**
3. 输入用户名、邮箱和密码

### 项目技术栈
- **后端**: Flask 3.0.0 + SQLAlchemy
- **前端**: Bootstrap 5 + Vanilla JavaScript
- **游戏**: HTML5 Canvas

### 常见问题

**Q: 游戏加载不出来？**
A: 确保服务器正在运行，检查浏览器控制台是否有错误信息。

**Q: 分数没有保存？**
A: 需要登录后才能保存分数。检查网络连接是否正常。

**Q: 可以添加新游戏吗？**
A: 可以。在 `config.py` 中添加游戏配置，在 `frontend/games/` 中创建游戏文件，然后在 `scripts/validate_games.py` 中验证。
