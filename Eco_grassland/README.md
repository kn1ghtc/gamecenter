# 小羊吃草生态模拟游戏 - 项目结构说明

## 项目概述
一款基于 Python 和 Pygame 的生态模拟游戏，展示动物与环境的相互作用。

## 优化后的项目结构

```
Eco_grassland/
├── README.md              # 项目说明文档
├── main.py                # 主游戏入口
├── modern_tools.py        # 现代化工具模块（统一配置和资源管理）
├── game_entities.py       # 游戏实体定义（动物、草地等）
├── ecosystem.py           # 生态系统核心逻辑
├── ui_manager.py          # 用户界面管理
└── camera.py              # 摄像机系统
```

## 模块说明

### 1. `main.py` - 主游戏类
- **EcoGrasslandGame**: 游戏主控制类
- 负责游戏循环、事件处理、渲染管理
- 统一游戏状态管理和用户交互

### 2. `modern_tools.py` - 现代化工具模块
**核心功能：**
- 解决 `pkg_resources` 弃用警告问题
- 提供现代化的资源管理方案
- 统一管理游戏常量配置

**主要组件：**
- `ModernResourceManager`: 使用 `importlib.resources` 的现代资源管理器
- 环境配置函数：自动设置 pygame 环境，避免警告
- 游戏常量：屏幕尺寸、地图尺寸等配置

### 3. `game_entities.py` - 游戏实体
**定义的实体类：**
- `GrassState`: 草地状态枚举
- `AnimalType`: 动物类型枚举
- `AnimalState`: 动物状态枚举
- `GrassTile`: 草地瓦片类
- `Animal`: 动物基类及其行为

### 4. `ecosystem.py` - 生态系统
- **EcoSystem**: 生态系统核心管理类
- 处理动物生命周期、食物链关系
- 管理环境压力和生态平衡
- 渲染生态元素

### 5. `ui_manager.py` - 用户界面
- **UIManager**: 用户界面管理器
- 处理控制面板、统计显示
- 管理用户输入和界面交互

### 6. `camera.py` - 摄像机系统
- **Camera**: 摄像机控制类
- 处理视角移动、缩放
- 坐标系转换

## 🎮 游戏特色

### 🌿 真实的生态循环
- **动态草地系统**：草地会根据环境压力和动物活动自然枯萎和重生
- **营养循环**：动物粪便为土壤提供营养，促进草地恢复
- **环境压力**：过度放牧会导致生态系统退化

### 🦌 智能动物行为
- **基础需求**：所有动物都有饥饿、口渴、疲劳等基本需求
- **行为状态机**：动物会根据需求智能地觅食、饮水、休息
- **种群特性**：
  - **羊 🐑**：温和食草动物，喜欢群居，移动速度中等
  - **兔子 🐰**：繁殖快，食量小，移动灵活
  - **牛 🐄**：食量大，移动缓慢，对草地影响显著

### 🎯 玩家互动系统
- **动物添加**：左键点击添加选中类型的动物
- **信息查看**：右键查看动物和草地的详细状态
- **视角控制**：自由移动和缩放观察整个生态系统
- **时间控制**：可以暂停、加速观察生态变化

### 📊 可视化数据统计
- **实时统计**：动物数量、生态压力、草地状况
- **健康指标**：承载能力、枯萎草地比例
- **动态图表**：生态系统健康度可视化

## 🎨 游戏机制详解

### 🌱 草地生态系统
- **4种草地状态**：枯萎 → 稀疏 → 正常 → 茂盛
- **动态再生**：草地会根据环境条件自然恢复
- **营养循环**：动物活动影响草地品质

### 🦌 动物行为AI
- **需求驱动**：基于饥饿、口渴、疲劳的状态机
- **群体行为**：动物会寻找同类聚集
- **环境感知**：智能寻找食物、水源和休息地

### ⚖️ 生态平衡机制
- **承载能力**：每200个草地瓦片支持1只动物
- **生态压力**：动物密度和草地枯萎率的综合指标
- **反馈调节**：高压力环境下动物能量消耗增加

### 📈 数据可视化
- **实时统计**：动物数量、生态健康度
- **趋势分析**：草地状况变化趋势
- **警报系统**：生态压力过高时的视觉提醒

## 🐛 故障排除

### 常见问题

1. **游戏启动失败**
   - 确保已安装 Pygame：`pip install pygame`
   - 检查 Python 版本（需要 3.7+）

2. **性能问题**
   - 降低动物数量
   - 关闭调试模式（F3）
   - 减少地图大小

3. **UI 显示异常**
   - 检查屏幕分辨率设置
   - 尝试重新启动游戏
