# 俄罗斯方块游戏 - Enhanced Edition

一个使用Pygame开发的增强版俄罗斯方块游戏，具有精美的3D效果、粒子特效和100关卡系统。

## ✨ 功能特点

### 核心特性
- **100关卡系统** - 从简单到极致挑战，关卡越高速度越快
- **智能分数系统** - 单行、双行、三行、四行消除不同得分，支持连击加分
- **自适应屏幕** - 根据屏幕分辨率自动调整窗口大小和方块尺寸
- **幽灵方块** - 实时显示方块将要落在哪里，便于精确操作

### 视觉效果
- **3D方块渲染** - 渐变色、高光、阴影效果，视觉更立体
- **粒子特效** - 消除方块时的爆炸粒子效果
- **动画效果** - 消行动画、升级通知、连击提示
- **动态背景** - 网格动画背景，增强游戏氛围
- **精美UI** - 半透明面板、渐变边框、现代化设计

### 游戏系统
- **连击系统** - 连续消除行数越多，得分倍数越高
- **Wall Kick** - SRS旋转系统，支持墙踢操作
- **硬下落 / 软下落** - Space键瞬间下落，方向键↓缓慢下落
- **暂停功能** - 随时暂停游戏
- **快速重启** - 按R键快速重新开始

### 音效系统
- **背景音乐** - 循环播放的背景音乐
- **音效反馈** - 移动、旋转、消除、升级、游戏结束等音效
- **音量控制** - 独立的音乐和音效音量控制
- **自动资源下载** - 首次运行自动下载字体和音效资源

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| ← / → | 左右移动方块 |
| ↓ | 软下落（缓慢下落，每格得1分） |
| Space | 硬下落（瞬间落到底部，每格得2分） |
| ↑ / Z | 顺时针旋转方块 |
| P | 暂停/继续游戏 |
| R | 重新开始游戏 |
| Esc | 退出游戏 |

## 🎯 评分规则

### 基础得分
- **单行消除**: 100分
- **双行消除**: 300分  
- **三行消除**: 500分
- **四行消除（Tetris）**: 800分

### 额外得分
- **软下落**: 每格1分
- **硬下落**: 每格2分
- **连击加成**: 连续消除时得分 × (1 + (连击数-1) × 0.5)
- **关卡倍数**: 最终得分 × 当前关卡数

### 升级条件
- 每消除10行升一级
- 最高100级
- 关卡越高，方块下落速度越快

## 📦 安装说明

### 环境要求
- Python 3.12+
- Windows 11（或其他支持Pygame的系统）
- 至少800x600分辨率屏幕

### 安装依赖

```bash
pip install pygame
```

### 运行游戏

```bash
# 方式1: 直接运行主文件
cd d:\pyproject\gamecenter\tetris
python main.py

# 方式2: 作为模块运行（从pyproject根目录）
cd d:\pyproject
python -c "from gamecenter.tetris.main import main; main()"
```

## 🎨 技术实现

### 关卡系统
```python
# 关卡速度计算公式
fall_speed = max(100, 1000 - (level - 1) * 15)  # 毫秒
```
- 第1关：1000ms（1秒）
- 第50关：265ms
- 第100关：100ms（最快）

### 分数计算
```python
# 连击分数公式
combo_multiplier = 1.0 + (combo_count - 1) * 0.5
final_score = base_score * combo_multiplier * level
```

### 3D渲染效果
- 使用渐变色模拟光照
- 左上角高光增强立体感
- 右下角阴影增加深度
- 多层边框创建3D边缘

### 粒子系统
- 消行时生成粒子爆炸效果
- 粒子具有重力和生命周期
- 透明度随时间衰减
